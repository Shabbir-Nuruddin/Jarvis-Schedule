Desktop App Instructions

To run this as a proper desktop app (its own window, taskbar icon, no browser tabs):

Electron App (VS Code — recommended):

1. Install Node.js from nodejs.org (free, one-time)
2. Put all downloaded files in ONE folder on your desktop
3. Open that folder in VS Code
4. Open the VS Code terminal (Ctrl + `)
5. Run: npm install
6. Then run: npm start
7. Your app opens in its own window — no browser, no tabs.

To launch it again later: open VS Code → terminal → npm start

To run Jarvis Focus Guardian:
In VS Code terminal: pip install SpeechRecognition pyaudio pyautogui google-generativeai win10toast pygetwindow
Then: python jarvis_guardian.py
